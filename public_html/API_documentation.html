API Online Documentation

URIs relative to http://appserver.alunos.di.fc.ul.pt/~aw003/rest/

Method HTTP request Description
articlesOrder GET /articlesOrder Get the top-n related articles for a given disease.
articlesInfo GET /articlesInfo Get the top-n related links and titles for a given disease.
articlesAbstracts GET /articlesAbstracts Get the top-n related abstracts for a given disease.
relatedDiseases GET /relatedDiseases Get the top-n related diseases for a given disease.
tweets GET /tweets Get the top-n most recent tweets for a given disease.
photos GET /photos Get the top-n most recent photos for a given disease.
metadata GET /metadata Get metadata for a given disease.
diseases GET /diseases Get the list of diseases whose label match a given prefix.
articlesMER GET /articlesMER Get the diseases mentioned in a given article and where in the text they are mentioned.
articleFeedbackPos POST /articleFeedbackPos Update the explicit positive feedback of the relevance of an article for a given disease.
articleFeedbackNeg GET /articleFeedbackNeg Update the explicit negative feedback of the relevance of an article for a given disease.
